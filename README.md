# LP01

A React TypeScript project built with Vite.

## Features

- React 18
- TypeScript
- Vite
- TailwindCSS
- ESLint

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## Technologies Used

- React
- TypeScript
- Vite
- TailwindCSS
- ESLint
